## 🕹️ How to Play
- Use your **arrow keys** ⬆️⬇️⬅️➡️ to move the tiles.  
- When two tiles with the same number touch, they **merge** into one.  
- Keep going until you reach the **2048 tile** 🏆  
- Game over if there are no more moves left!
